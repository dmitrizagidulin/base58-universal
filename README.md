## Install

- Node.js 14+ is recommended.

### NPM

To install via NPM:

```
npm install @interop/base58-universal
```

### Development

To install locally (for development):

```
git clone https://github.com/interop-alliance/base58-universal.git
cd base58-universal
npm install
```
